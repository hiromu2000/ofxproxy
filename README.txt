OFXProxy:対応金融機関のOFXファイルをダウンロードするサービス

OFXProxyは、対応金融機関の口座情報をOFXファイルとしてダウンロードできるサービスです。取得した明細データは、OFX対応ソフト・サービスに取り込めます。

OFXProxyを実行するには、PHPバージョン5.4以降が必要です。
